Implict wait is used to improve the stability of test scripts
by allowing the selenium to wait for a certain period of time
before performing actions on elements in a web application

Whenever your automation script tries to interact with an element like, click method, send keys method, etc.
if the element is not immediately present in the DOM,
selenium will wait for the specified amount of time before giving up and throwing an exception. During this wait period,
selenium will repeatedly check for the presence of the element.
